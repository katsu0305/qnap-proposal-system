## 📊 見積もり基本構成

### 見積もり項目の標準構成

```
【基本構成】
1. ハードウェア
   ├─ NAS本体
   ├─ 内蔵HDD/SSD
   └─ オプション（メモリ、ネットワークカード等）

2. 保守サービス
   ├─ 標準保証（無料）
   ├─ 延長保証
   ├─ センドバック保守
   ├─ オンサイト保守
   └─ 24h365d保守

3. 導入支援
   ├─ 初期設定作業
   ├─ データ移行支援
   ├─ ユーザー教育
   └─ ネットワーク設定

4. 周辺機器
   ├─ UPS（無停電電源装置）
   ├─ ラックマウント金具
   ├─ ネットワークスイッチ
   └─ ケーブル類

5. ライセンス（必要に応じて）
   ├─ QVR Pro（監視カメラ）
   ├─ QuMagie（AI写真管理）
   └─ その他アプリ
```

## 💸 割引率ガイドライン

### 基本的な割引ルール

```
【原則】
✓ 初回提示価格 = 標準価格（割引前提なし）
✓ 値引き交渉には必ず条件付与
✓ 最大割引率を超えないこと
✓ 割引後も最低粗利率を確保
```

### 案件規模別の最大割引率

| 総額 | 最大割引率 | 条件 |
|------|-----------|------|
| ～50万円 | **5%** | 即決、紹介案件獲得 |
| 50～100万円 | **8%** | 即決、次回案件確約 |
| 100～200万円 | **10%** | 複数台、年間保守契約 |
| 200～500万円 | **12%** | 複数拠点、長期保守契約 |
| 500万円～ | **15%** | 大型案件、グループ展開 |

### 割引条件の例

| 条件 | 割引率 | 備考 |
|------|-------|------|
| **即決割引** | 2～3% | 見積もり提示から1週間以内の発注 |
| **複数台割引** | 3～5% | 2台以上の同時購入 |
| **紹介割引** | 2～3% | 他社紹介確約 |
| **年間保守契約** | 5～8% | 3年以上の保守契約セット |
| **既存顧客割引** | 3～5% | 過去取引実績あり |
| **グループ割引** | 5～10% | 同一グループ内複数拠点 |
| **教育機関割引** | 5～10% | 学校、教育機関向け |
| **官公庁入札** | 10～15% | 入札案件（最低粗利率死守） |

### 割引シミュレーション

#### 例1: 中規模オフィス（TS-433eU）

```
標準価格: ¥600,000（税抜）
├─ ハードウェア: ¥480,000（粗利率25%）

→ 最低ライン（条件付きで可）
```

## 🎯 まとめ: 見積もり作成の鉄則

### 5つの基本原則

```
1. 【顧客第一】
   顧客の課題解決を最優先に考える
   価格だけでなく、価値を提案する

2. 【適正利益】
   目標粗利率を達成する
   最低粗利率（15%）は死守する

3. 【透明性】
   価格の根拠を明確に説明する
   隠れコストを作らない

4. 【Win-Win】
   顧客も自社も満足する取引を目指す
   過度な値引きで信頼を失わない

5. 【継続性】
   1回の取引で終わらせない
   長期的な関係構築を重視する
```

### 避けるべき行動

```
❌ 最初から最安値を提示する
❌ 根拠なく値引きする
❌ 保守サービスを軽視する
❌ 導入支援を無償で提供する
❌ 粗利率を度外視する
❌ 他社を批判する
❌ 嘘をつく・誇張する
❌ 約束を守らない
```

### 成功のポイント

```
✅ ヒアリングを丁寧に行う
✅ 顧客の課題を正確に理解する
✅ 最適な構成を提案する
✅ 価値を明確に説明する
✅ 複数の選択肢を提示する
✅ 誠実に対応する
✅ 迅速に対応する
✅ アフターフォローを徹底する
```
