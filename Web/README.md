# QNAP NAS ソリューション提案サイト

QNAP NAS製品の提案をサポートする営業支援Webサイトです。

## 機能

- **製品選定ツール**: 質問に答えるだけで最適な製品を提案
- **製品比較**: 複数製品のスペック比較
- **業種別ソリューション**: 10業種の導入パターン
- **見積もり作成**: その場で見積書を作成
- **導入事例**: 実際の導入事例を掲載
- **技術資料**: RAIDやバックアップの解説

## 使い方

1. このリポジトリをクローンまたはダウンロード
2. `index.html` をブラウザで開く
3. すぐに閲覧可能（サーバー不要）

## ファイル構成

```
Web/
├── index.html           # トップページ
├── selection.html       # 製品選定ツール
├── comparison.html      # 製品比較
├── solutions.html       # 業種別ソリューション
├── estimate.html        # 見積もり作成
├── cases.html          # 導入事例
├── technical.html      # 技術資料
├── css/
│   └── style.css       # スタイルシート
├── js/
│   └── selection.js    # 製品選定ロジック
└── images/            # 画像ファイル
```
